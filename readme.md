# 🐾 PetVibe - Landing Page Profissional

O **PetVibe** é uma landing page moderna, responsiva e funcional desenvolvida para um Pet Shop. Este projeto foi criado com o objetivo de consolidar conhecimentos fundamentais de desenvolvimento web front-end, focando em semântica, design moderno e interatividade.

---

## 🚀 Tecnologias Utilizadas

Para garantir performance e um código limpo, o projeto utiliza exclusivamente tecnologias nativas:

* **HTML5:** Estruturação semântica para melhor SEO e acessibilidade.
* **CSS3:** Layouts avançados utilizando **Flexbox** e **CSS Grid**, além de variáveis nativas para fácil manutenção.
* **JavaScript (ES6):** Manipulação do DOM para validação de formulários e interatividade.
* **Google Fonts:** Utilização da fonte 'Poppins' para uma leitura moderna e acolhedora.

---

## 🛠️ Funcionalidades e Diferenciais

* **Design Responsivo:** A página se adapta perfeitamente a dispositivos móveis, tablets e desktops.
* **Scroll Inteligente:** Navegação suave via âncoras com ajuste de compensação para o header fixo.
* **Interface Moderna:** Uso de técnicas de sobreposição de elementos (`position: absolute/relative`) e efeitos de hover.
* **Formulário de Contato:** Sistema de validação de campos via JavaScript com feedback em tempo real para o usuário.
* **Seção Sobre Padronizada:** Indicadores de desempenho e história da marca integrados ao layout.
